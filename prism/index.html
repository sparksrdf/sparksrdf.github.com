Create a PRISM, filter you data.

Simple as SPARQL and JQuery

Sparks Prism combines the simplicity of JQuery with the SPARQL syntax for creating views and controls that enable your application to deal with RDF repositories.

Keep Everything in Sync

Prism uses a subset of the SPARQl vocabulary for filtering your data, each filter (lens) stay synchronised with each other. If a filter modify your prism, all the connected filters and views get notified of the changes.

Get Creative

Reuse, modify and create new lenses for you own application. Combine lenses together and customise them using plain CSS.

Checkout the WeKnowIt Explorer and Topica for some insights on what you can do with Prism.
