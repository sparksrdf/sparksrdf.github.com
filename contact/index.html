If you have a question or query that is not answered elsewhere on this website, you can contact us.

User help

The documentation is not ready at the moment. If you have still some questions after studying the code and the demo file, you can ask your question on the user mailing list.

Development

The best way to participate in the development of Sparks is to fork the project. You can also subscribe to the developer mailing list: sparksrdf-dev@googlegroups.com.

Website

If you have discovered a problem with the web-site, or wish to help improve our web-pages, you can fork the Sparks website and submit a merge request.
